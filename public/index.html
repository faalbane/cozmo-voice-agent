Key Features

Built to meet every requirement of a production voice AI system

☎
SIP Telephony

Twilio SIP trunk integration via LiveKit's native SIP bridge. Accept inbound PSTN calls with automatic room creation and agent dispatch.

⚡
Barge-In Support

Interruptible TTS with configurable speech detection. Agent stops speaking and processes new input when caller interrupts mid-sentence.

🧠
LLM-Powered Dialog

GPT-4o-mini with function calling for knowledge base lookups. Natural turn-taking with Silero VAD endpointing.

📚
Knowledge Base

ChromaDB vector search over company documents. Agent answers policy questions accurately using retrieval-augmented generation.

📈
Full Observability

Prometheus metrics with Grafana dashboards. Per-turn latency breakdown (STT/LLM/TTS), MOS estimation, jitter, packet loss tracking.

🛡
Resilience

LLM timeout fallback, circuit breaker pattern, health check endpoints. Subprocess isolation — one crashed call doesn't affect others.

Scaling Strategy

From development to 1,000+ concurrent calls

1-10 Calls

Single host, Docker Compose, 1 agent worker. Development and demo setup.

10-50 Calls

Single host, 3 agent workers scaled via Docker Compose. Load-aware dispatch via LiveKit.

50-100 Calls

Multi-host, 5-7 workers. LiveKit's built-in load_fnc balances call distribution automatically.

100-1,000 Calls

Kubernetes with HPA, LiveKit Cloud, managed Redis. 50-70 workers with auto-scaling on concurrent call count.
